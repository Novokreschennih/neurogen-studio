# 🔐 NeuroGen Studio — Настройка API ключей

## Поддерживаемые провайдеры

NeuroGen Studio поддерживает **три провайдера** AI моделей:

### 1. Google Gemini 💎

**Провайдер по умолчанию**

#### Как получить ключ:
1. Перейдите на [Google AI Studio](https://aistudio.google.com/apikey)
2. Войдите через Google аккаунт
3. Нажмите "Create API Key"
4. Скопируйте ключ

#### Преимущества:
- ✅ Бесплатный тариф с лимитами
- ✅ Высокая скорость генерации
- ✅ Поддержка мультимодальных моделей
- ✅ Прямая интеграция без посредников

#### Модели:
- `gemini-2.5-flash` — Быстрая модель (рекомендуется)
- `gemini-3-pro-preview` — Продвинутая модель

---

### 2. OpenRouter 🌐

**Агрегатор AI моделей**

#### Как получить ключ:
1. Перейдите на [OpenRouter.ai](https://openrouter.ai)
2. Зарегистрируйтесь
3. Перейдите в раздел "Keys"
4. Создайте новый API ключ
5. Пополните баланс (от $5)

#### Преимущества:
- ✅ Доступ к множеству моделей (Claude, GPT-4, Llama и др.)
- ✅ Оплата только за использование
- ✅ Единый API для всех провайдеров
- ✅ Подробная статистика расходов

#### Популярные модели:
- `anthropic/claude-3.5-sonnet` — Лучшая для кода
- `openai/gpt-4o` — Универсальная модель
- `google/gemini-flash-1.5` — Быстрая генерация
- `meta-llama/llama-3.1-70b-instruct` — Open-source

---

### 3. RouterAI 🔄

**Альтернативный агрегатор**

#### Как получить ключ:
1. Перейдите на [RouterAI.ai](https://routerai.ai)
2. Зарегистрируйтесь
3. Перейдите в раздел API
4. Создайте новый ключ

#### Преимущества:
- ✅ Конкурентные цены
- ✅ Поддержка популярных моделей
- ✅ Быстрая техподдержка
- ✅ Удобный интерфейс

#### Популярные модели:
- `openai/gpt-4o` — Флагманская модель
- `openai/gpt-4o-mini` — Лёгкая версия
- `anthropic/claude-3-5-sonnet-20240620` — Claude 3.5

## Настройка в приложении

### Способ 1: Через интерфейс (рекомендуется)

1. Откройте NeuroGen Studio
2. Нажмите на иконку ⚙️ (Настройки) в верхней панели
3. Выберите вкладку провайдера (Gemini / OpenRouter / RouterAI)
4. Введите API ключ
5. Нажмите "Сохранить"

### Способ 2: Через экран входа

1. При первом запуске нажмите "API ключи"
2. Введите один или несколько ключей
3. Нажмите "Сохранить и продолжить"

### Способ 3: Через консоль браузера

Откройте консоль разработчика (F12) и выполните:

```javascript
// Google Gemini
localStorage.setItem('neurogen-gemini-key', 'ваш-ключ-gemini');

// OpenRouter
localStorage.setItem('neurogen-openrouter-key', 'ваш-ключ-openrouter');

// RouterAI
localStorage.setItem('neurogen-routerai-key', 'ваш-ключ-routerai');

// Обновите страницу
location.reload();
```

## Хранение ключей

Ключи хранятся в **localStorage** браузера:

| Ключ | localStorage ключ |
|------|------------------|
| Gemini | `neurogen-gemini-key` |
| OpenRouter | `neurogen-openrouter-key` |
| RouterAI | `neurogen-routerai-key` |
| Активный провайдер | `neurogen-provider` |

### Безопасность:
- ✅ Ключи хранятся только в вашем браузере
- ✅ Не передаются на серверы NeuroGen
- ✅ Используются только для запросов к AI провайдерам
- ✅ Можно очистить в любой момент через настройки браузера

## Переключение между провайдерами

### В настройках:
1. Откройте ⚙️ Настройки
2. Выберите нужную вкладку (Gemini / OpenRouter / RouterAI)
3. Введите ключ если ещё не введён
4. Выберите модель из списка
5. Нажмите "Сохранить"

### В модулях:
Каждый модуль (Конструктор, Редизайн, Деплой) имеет свой селектор провайдера:
- Выберите провайдера из выпадающего списка
- Введите модель (если требуется)
- Нажмите "Генерировать"

## Решение проблем

### "API key not configured"
- Проверьте, что ключ сохранён в настройках
- Обновите страницу (F5)
- Проверьте консоль браузера на ошибки

### "Invalid API key"
- Проверьте правильность ключа (без пробелов)
- Убедитесь, что ключ активен в личном кабинете провайдера
- Проверьте баланс (для платных тарифов)

### "Rate limit exceeded"
- Вы превысили лимит запросов
- Подождите несколько минут
- Или переключитесь на другого провайдера

### "Insufficient credits"
- Пополните баланс в личном кабинете провайдера
- Проверьте статус подписки

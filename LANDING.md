# 🎨 Лендинг NeuroGen Studio

## Обзор

Лендинг NeuroGen Studio — это продающая страница приложения, которая показывает все преимущества и выгоды для пользователей.

## Расположение

- **Исходный файл:** `public/landing.html`
- **Генератор:** `src/utils/landingGenerator.ts`

## Структура лендинга

### 1. Hero Section (Главный экран)
- Заголовок с градиентным текстом
- Подзаголовок с описанием
- Две кнопки CTA (Call-to-Action)
- Статистика (4 карточки)

### 2. Features Section (Возможности)
- 3 карточки модулей:
  - 🧲 Конструктор лендингов
  - 🎨 Редизайн сайта
  - 🚀 Деплой на GitHub

### 3. Benefits Section (Преимущества)
- 6 карточек с ключевыми выгодами:
  - ⚡ Быстрый запуск
  - 💰 Экономия бюджета
  - 🎯 Без технических знаний
  - 🔄 Гибкость
  - 🤖 Мульти-модельность
  - 🔒 Безопасность

### 4. CTA Section (Призыв к действию)
- Финальный призыв к запуску приложения
- Кнопка "Запустить NeuroGen Studio"

### 5. Footer (Подвал)
- Логотип и название
- Копирайт

## Дизайн

### Цветовая схема
- **Фон:** `#05050a` (тёмный)
- **Градиент:** Purple → Cyan → Pink
- **Акценты:** 
  - Cyan: `#06B6D4`
  - Purple: `#A855F7`
  - Pink: `#EC4899`

### Эффекты
- **Glassmorphism:** Полупрозрачные карточки с blur
- **Glow:** Светящиеся эффекты
- **Gradient Text:** Анимированный градиент на тексте
- **Float Animation:** Плавающие элементы

### Шрифты
- **Основной:** Inter (Google Fonts)
- **Размеры:** 300-800 weight

## Функциональность

### Кнопка "Запустить бесплатно"
```javascript
function startApp() {
    const token = localStorage.getItem('ng_token');
    const isAuthenticated = localStorage.getItem('isAuthenticated');
    
    if (token && isAuthenticated === 'true') {
        window.location.href = '/'; // Открываем приложение
    } else {
        if (window.ng_showLogin) {
            window.ng_showLogin(); // Показываем форму входа
        }
    }
}
```

### Интеграция с auth.js
- Проверяет токен аутентификации
- Показывает форму входа если не авторизован
- Переключает на приложение после успешного входа

## Адаптивность

Лендинг полностью адаптивен:
- **Mobile:** < 640px
- **Tablet:** 640px - 1024px
- **Desktop:** > 1024px

Используется Tailwind CSS с брейкпоинтами:
- `sm:` 640px
- `md:` 768px
- `lg:` 1024px
- `xl:` 1280px

## Как использовать

### Вариант 1: Через public/landing.html
Просто откройте файл в браузере или через веб-сервер.

### Вариант 2: Через генератор
```typescript
import { generateLandingHTML } from './utils/landingGenerator';

const html = generateLandingHTML();
// Используйте html для рендеринга
```

### Вариант 3: Автоматически
Лендинг автоматически показывается при первом посещении сайта (если нет токена аутентификации).

## Редактирование

Для изменения контента отредактируйте `public/landing.html` или обновите шаблон в `src/utils/landingGenerator.ts`.

### Изменение цветов
В секции `<script>` с конфигом Tailwind:
```javascript
tailwind.config = {
    theme: {
        extend: {
            // Измените цвета здесь
        }
    }
}
```

### Добавление новой секции
Добавьте новый `<section>` после существующих секций в `landing.html`.
